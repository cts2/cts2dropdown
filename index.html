CTS2 Dropdown

A Javascript widget for populating dropdown menus with CTS2 Vocabulary content.

View project onGitHub

CTS2 Dropdown allows you to add semantically meaningful and standard content to your dropdown lists.

Dwelling Type: 


Geographic Location:

Usage

Add this script to the <head> section: 
<script type="text/javascript" src="https://raw.github.com/cts2/cts2dropdown/master/cts2dropdown.js"></script>
Add a cts2-valueset class to your <select> dropdown lists: 
<select class="cts2-valueset(name : PHVS_DwellingType_NND)"></select>

<select class="cts2-valueset(name : PHVS_GeographicLocationHistory_CDC)"></select>
